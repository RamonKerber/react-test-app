# Aplicação de Teste com React 🚀

Este projeto foi desenvolvido como atividade da disciplina de desenvolvimento de aplicações.

## Descrição
A aplicação foi criada utilizando React e possui um contador de cliques interativo.

## Funcionalidades
- Interface simples
- Contador de cliques
- Mensagem exibida após vários cliques

## Tecnologias utilizadas
- React
- JavaScript
- Vite

## Como executar o projeto

1. Instalar as dependências:


npm install


2. Rodar o projeto:


npm run dev


Depois abra no navegador o endereço exibido no terminal.
